# Barber Scheduler

**Barber Scheduler** é uma aplicação de agendamento de barbearia desenvolvida em Java com **Swing** e **SQLite** para persistência de dados. Permite aos utilizadores registarem-se, fazerem login e agendarem horários com barbeiros.

## Funcionalidades

- **Login e Registo** de utilizadores
- **Gestão de Barbeiros**
- **Agendamentos** de horários
- **Interface Gráfica** com Java Swing

## Tecnologias

- **Swing** (UI)
- **SQLite** (DB)
- **JDBC** (conexão com DB)
